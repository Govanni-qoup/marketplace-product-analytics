# Marketplace Product Analytics

Учебный проект по продуктовой аналитике маркетплейса.

## Стек

- SQL
- PostgreSQL
- Python
- Pandas
- Matplotlib

## Анализ

- Funnel Analysis
- Retention
- RFM Segmentation
- Session Analysis
- A/B Testing

## Цель

Оценить влияние продуктовых изменений на конверсию, выручку и удержание пользователей.
